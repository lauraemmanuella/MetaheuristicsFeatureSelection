# MetaheuristicsFeatureSelection
Metaheurísticas bioinspiradas para seleção de atributos em bases de dados de Aprendizado de Máquina

Metaheurísticas implementadas: Algoritmo Genético (GA) e Otimização por Nuvem de Partículas (PSO)

Problema: Seleção de atributos em bases de dados de Aprendizado de Máquina
